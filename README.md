Overview :
This assignment, titled Final Visualisation Report, tells the story of climate change through data. Using the climate_change_dataset-2.csv dataset, the report explores how key climate variables have shifted over time, how rainfall is distributed across the data, how climate indicators relate to one another, and how renewable energy adoption varies between countries. The report is 400–600 words, submitted as a PDF, and all figures and tables are produced in Python using numpy, pandas, matplotlib, and seaborn.

The Dataset:
The dataset used is climate_change_dataset-2.csv, which contains country-level and time-series data on a range of climate change indicators — including temperature, rainfall, CO2 emissions, and renewable energy usage across multiple countries and years.

The Plots and Table:
There are four visualisation elements in total, each one adding something new to the story.
The summary statistics table covering the key numeric variables in the dataset — things like mean, standard deviation, min, and max. This is the first thing the reader sees and gives an immediate sense of the scale and spread of the data before any trends are discussed.The first plot is a line plot titled Climate Change Trends, with year on the x-axis and value on the y-axis. This is the core of the story — it shows how the key climate variables have changed over time.The second is a histogram showing rainfall distribution, with rainfall in mm on the x-axis and frequency on the y-axis. Rather than just reporting average rainfall, this plot shows the full shape of the distribution. The third is a heatmap of the correlation between the climate change variables. This sits alongside the line plot to explain why certain trends move together. The fourth is a box plot titled Renewable Energy Usage across Countries, with renewable energy percentage on the x-axis and country on the x-axis.

How to Run It:
Make sure pandas, numpy, matplotlib, and seaborn are installed, then run the Python script or open the Jupyter notebook and execute all cells. The plots will save as image files which were then imported into the Word document and exported as the final PDF report.
